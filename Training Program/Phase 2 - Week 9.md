# Phase 2: Strength-Speed (Weeks 9–16)

## Phase 2 Goals

- Convert positional improvements from Phase 1 into heavier loads
- Push squat toward 195-200 range (triples → doubles)
- Snatch should be hitting 110-115 regularly by end of phase
- Clean & Jerk now combined — practice the full lift under fatigue
- Jerk built independently in Phase 1, now integrated
- Continue knee rehab (should be noticeably better by now)
- Gymnastics: ring MU attempts, HS walk first steps

## Phase 2 Weekly Template

| Day | Primary | Secondary | Accessory (15-20 min) |
|---|---|---|---|
| Monday | Snatch (heavy singles/doubles) | Back squat (triples/doubles) | Unilateral + thoracic |
| Tuesday | Clean & Jerk complexes | Push press / Strict press | Ring MU skill + front rack mobility |
| Wednesday | Clean & deadlift | Metcon (10-15 min) | Unilateral leg work |
| Thursday | Front squat (heavy) + Jerk from rack | Metcon (8-12 min) | Thoracic + shoulder |
| Friday | Snatch (daily max) + OHS | HS walk skill | Hip mobility + knee prehab |

> [!tip] Phase 2 Coaching Cues
> - Intensity is climbing — prioritize quality singles over grinding reps
> - "Make weight" mentality: if a rep is ugly, don't go heavier. Repeat the weight until it's clean.
> - The snatch daily max on Friday is a DAILY max, not an all-time max. Some days 90% feels like a max. That's fine.

## Week 9 — Phase 2 Start

> [!info] Week 9 Targets
> Oly intensity: 80-85% | Squat intensity: 78-80% | Focus: transition to heavier loads, lower reps. Use updated % bases from Week 8 test.

> [!warning] Update Your Bases

### Monday — Snatch + Back Squat

**Warm-up (10 min):** Thoracic foam roll, banded pull-aparts, snatch grip BTN press, OHS empty bar, snatch balance x5

**Primary: Snatch**
- Hang snatch (above knee) + Snatch from floor: 5 x (1+1) @ 80%
- Rest 2:00-2:30
- Focus: speed under the bar, aggressive lockout

**Secondary: Back Squat**
- 5 x 3 @ 80%
- No tempo — controlled descent, strong drive

**Accessory Window (15-20 min):**
- Peterson step-downs: 3 x 10 each (with DB)
- Spanish squat iso hold: 3 x 30 sec

### Friday — Snatch Daily Max + OHS + HS Walk

**Warm-up (10 min):** Snatch grip RDL x5, hang snatch high pull x5, snatch balance x5

**Primary: Snatch — Work to Daily Max**
- Build up in planned jumps: empty → 50% → 60% → 70% → 80% → 85% → daily max
- Then back off: 3 x 1 @ 80%
- Daily max = heaviest clean single today. Not an all-time max attempt.

**Secondary: Overhead Squat**
- 4 x 3 @ 70% of snatch

**Gymnastics Skill: HS Walk (5-10 min)**
- Wall HS hold: 2 x 30 sec
- HS walk off wall: 5 attempts (aim for 3-5 steps)
- If you can take steps: practice 3m distances x 3 attempts

**Accessory Window (15-20 min):**
- Peterson step-downs: 3 x 10 each
- Pigeon stretch: 2 x 60 sec each
- 90/90 hip rotation: 2 x 10 each
